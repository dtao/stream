      ctrl+enterinserts a new element before the current one
      ctrl+shift+enterinserts a new element at the end of the article
      entercreates a new element
      backspacedeletes the current element (if empty)
      ctrl+1changes the current element to a top-level heading (<h1>)
      ctrl+2changes the current element to a subheading (<h2>)
      ctrl+3changes the current element to a secondary subheading (<h3>)
      ctrl+pchanges the current element to a paragraph (<p>)
      ctrl+lchanges the current element to a list item (<li>)
      ctrl+qchanges the current element to a blockquote (<blockquote>)
      ctrl+mcreates a code editor from the current element
      ctrl+dchanges the current element to a drawing area (<canvas>)
      ctrl+imakes the selected text italic
      ctrl+bmakes the selected text bold
      ctrl+`marks the selected text as source code (<code>)
      ctrl+kmarks the selected text as a keyboard key (<kbd>)
      ctrl+backspacemarks the selected text as deleted
      ctrl+aadd a hyperlink (<a>)
      ctrl+gadd a graphic (<img>)
      ctrl+=increases the width of the article
      ctrl+-decreases the width of the article
      ctrl+ssaves the article
